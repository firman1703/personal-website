my SKILL

HTML

HTML adalah bahasa markup standar yang digunakan untuk membuat struktur dasar sebuah halaman web. Anda dapat menggunakan HTML untuk membuat elemen-elemen seperti teks, gambar, tautan, formulir, dan banyak lagi. Ini adalah fondasi dari setiap halaman web.

CSS

CSS digunakan untuk mengontrol tata letak, tampilan, dan gaya dari elemen-elemen HTML di halaman web. Dengan CSS, Anda dapat mengatur warna, font, margin, padding, dan banyak lagi. Ini memungkinkan Anda untuk membuat halaman web yang menarik dan responsif.

JAVASCRIPT

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif. Dengan JavaScript, Anda dapat menanggapi tindakan pengguna, memvalidasi formulir, mengubah konten halaman secara dinamis, dan banyak lagi. Ini memungkinkan Anda untuk membuat aplikasi web yang dinamis dan kompleks.

BMI Calculator

aplikasi berbasis web yang dirancang untuk menghitung Indeks Massa Tubuh (BMI) seseorang. BMI adalah pengukuran yang umum digunakan untuk mengevaluasi apakah berat badan seseorang sesuai dengan tinggi badan mereka. Aplikasi ini memungkinkan pengguna untuk memasukkan berat badan dan tinggi badan mereka, dan kemudian menghitung nilai BMI berdasarkan rumus BMI standar
